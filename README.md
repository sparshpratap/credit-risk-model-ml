# Credit Risk Modeling using Machine Learning

## Project Overview
This project builds a machine learning model to classify loan applicants as Good or Bad credit risk.

The model was trained using the German Credit dataset and deployed as an interactive web application using Streamlit.

---

## Dataset
German Credit Dataset  
Source: https://www.kaggle.com/datasets/kabure/german-credit-data-with-risk

---

## Model
- Extra Trees Classifier
- Label Encoding for categorical variables
- Binary Classification

---

## Features Used
- Age
- Sex
- Job
- Housing
- Saving accounts
- Checking account
- Credit amount
- Duration
